Welcome!

In this example you will learn how to operate the debugger inside of the Chrome browser. Note that the process is almost identical for other browsers such as Firefox, however you may need to translate the process using your own knowledge into other browsers since menus, and buttons may be labled differently than Chrome.

Step 1

→ Let's first open up our inspector tools by right clicking and selecting "Inspect" at the very bottom of the context menu.

Step 2

→ After that you should see a panel appear on the left hand side, and a preselected tab called "Page."

Step 3

→ Use the left hand "Page" panel to find the JavaScript code attached to this page. Note that inline scripts will show up under the filename of this html file.
